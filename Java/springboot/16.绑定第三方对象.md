### 配置文件中数组和键值对的写法

```yaml
# List或者数组的写法 List<Server> servers
servers:
  - title: 服务器1
    ip: 12.21
  - title: 服务器2
    ip: 32.45

# Map<String, User> users
users:
  user1:
    name: 刘自航1
    sex: 男
    age: 22
  user2:
    name: 刘自航2
    sex: 女
    age: 24

```
