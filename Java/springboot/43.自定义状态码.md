### 使用ResponseEntity自定义状态码

```java
@GetMapping("/entity")
@ResponseBody
public ResponseEntity<User> returnEntity() {
    User user = new User();
    user.setName("lzh");
    user.setAge(22);

    ResponseEntity<User> userResponseEntity = new ResponseEntity<>(user, HttpStatus.OK);  // 这里自定义返回的状态码
    return userResponseEntity;
}
```
