### switch新特性

- 支持箭头写法

  ```java
  Interger a = 1;
  switch (a) {
  	case 0 -> sout("0");
      case 1 -> sout("1");
      default -> sout("3");
  }
  ```

- 支持yield返回参数

  ```java
  Interger a = 1;
  String s = switch (a) {
      case 0 : yield "0";
      case 1 : yield "1";
      default : yield "3";
  }
  ```

- 混合写法

  ```java
  Interger a = 1;
  String s = switch (a) {
      case 0 -> {
          sout("0");
          yield "0";
      } 
      case 1 -> {
          sout("1");
          yield "1";
      } 
      default -> {
          sout("3");
          yield "3";
      } 
  }
  ```

### 文本段特性

支持和python类似的三引号创建多行字符串

```java
String s = """
    java
    string
    """
```
