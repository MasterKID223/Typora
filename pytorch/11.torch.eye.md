### torch.eye

- 函数

  ```python
  torch.eye(n, m=None, *, out=None, dtype=None, layout=torch.strided, device=None, requires_grad=False) → Tensor
  ```

- 作用

  返回一个2维单位矩阵。

- 参数

  ```json
  n: '行数',
  m: '列数，默认是=n',
  out: '输出的向量',
  ```

- 例子

  ```shell
  >>> torch.eye(3)
  tensor([[ 1.,  0.,  0.],
          [ 0.,  1.,  0.],
          [ 0.,  0.,  1.]])
  ```
