### 与python性能测试对比

开一亿次根号。

mojo代码，耗时：2.86102294921875e-06

```python
from python import Python

def main():
    time = Python.import_module("time")
    st = time.time()
    print(st)
    res = 0
    for i in range(1, 100000000):
        res = math.sqrt(i)
    ed = time.time()
    print(ed)
    print(ed - st)
```
