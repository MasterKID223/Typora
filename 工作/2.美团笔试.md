#### 第二题

```
小美拿到了一个由正整数组成的数组，但其中有一些元素是未知的(用0来表示)现在小美想知道，如果那些未知的元素在区间[l, r],”范围内随机取值的话，数组所有无素之和的最小值和最大值分别是多少?共有q次询问。
输入描述：
第一行输入两个正整数n，1，代表数组大小和询问次数。
第二行输入n个整数ai，其中如果输入的ai为0，那么说明ai是未知的。
接下来的q行，每行输入两个正整数l, r，代表一次询问。
输出描述：
输出q行，每行输出两个正整数，代表所有元素之和的最小值和最大值。
示例：
输入：
3 2
1 0 3
1 2
4 4
输出：
5 6
8 8
说明：
只有第二个元素是末知的。
第一次询问，数组最小的和是 1+1+3=5，最大的和是 1+2+3=6。
第二次询问，显然数组的元素和必然为 8。
```

```java
import java.util.*;
import java.io.*;

class Main {
    
    static final int N = 100010;
    static int n, q;
    static int[] a = new int[N];
    static int[] pre = new int[N];
    
    
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String[] strs = br.readLine().split(" ");
        n = Integer.parseInt(strs[0]);
        q = Integer.parseInt(strs[1]);
        strs = br.readLine().split(" ");
        int cnt = 0;
        long sum = 0;
        for (int i = 0; i < n; i ++) {
            a[i] = Integer.parseInt(strs[i]);
            if (a[i] == 0) cnt ++;
            // if (i == 0) pre[0] = a[0];
            // else if (i >= 1) pre[i] = pre[i - 1] + a[i];
            sum += a[i];
        }
        while (q -- > 0) {
            strs = br.readLine().split(" ");
            int l = Integer.parseInt(strs[0]);
            int r = Integer.parseInt(strs[1]);
            long mi = 0, mx = 0;
            // mi = pre[n - 1] + cnt * l;
            // mx = pre[n - 1] + cnt * r;
            mi = sum + cnt * l;
            mx = sum + cnt * r;
            System.out.println(mi + " " + mx);
        }
    
    }
}
```

#### 第四题

```
我有拿到了一个大小为n的数组，我希望删除一个区间后，使得剩余所有元素的乘积末尾至少有K个 0。我想知道，一共有多少种不同的删除方案?
输入数据格式为：
第一行输入两个正整数n, k。
第二行输入n个正整数ai, 代表我拿到的数组。
1 <= n, k <= 10^5
1 <= ai <= 10^9
现在有一个样例：
样例输入为：
5 2
2 5 3 4 20
样例输出为：
4
对样例的解释：
第一个方案，删除[3]。
第二个方案，删除[4]。
第三个方案，删除[3,4]。
第四个方案，删除[2]。
一共有四种方案，所以样例输出为4。
```

```java
import java.util.*;
import java.io.*;

class Main {
    
    static final int N = 100010;
    static int n, k;
    static int[] a = new int[N];
    static int[] cnt2 = new int[N];
    static int[] pre2 = new int[N];
    static int[] cnt5 = new int[N];
    static int[] pre5 = new int[N];
    
    
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String[] strs = br.readLine().split(" ");
        n = Integer.parseInt(strs[0]);
        k = Integer.parseInt(strs[1]);
        strs = br.readLine().split(" ");
        for (int i = 0; i < n; i ++) {
            a[i] = Integer.parseInt(strs[i]);
            cnt2[i] = get2(a[i]);
            cnt5[i] = get5(a[i]);
            if (i == 0) {
                pre2[0] = cnt2[0];
                pre5[0] = cnt5[0];
            }
            else {
                pre2[i] = pre2[i - 1] + cnt2[i];
                pre5[i] = pre5[i - 1] + cnt5[i];
            }
        }
        // 枚举所有的区间情况
        int res = 0;
        for (int i = 0; i < n; i ++) {
            for (int j = i; j < n; j ++) {
                // 处理边界情况
                int num2 = 0, num5 = 0;
                if (i == 0) {
                    num2 = pre2[n - 1] - pre2[j];
                    num5 = pre5[n - 1] - pre5[j];
                }
                else if (i == n - 1) {
                    num2 = pre2[n - 2];
                    num5 = pre5[n - 2];
                }
                else if (j == n - 1) {
                    num2 = pre2[i - 1];
                    num5 = pre5[i - 1];
                }
                else {
                    num2 = pre2[n - 1] - pre2[j] + pre2[i - 1];
                    num5 = pre5[n - 1] - pre5[j] + pre5[i - 1];
                }
                int mi = Math.min(num2, num5);
                if (mi >= k) res ++;
            }
        }
        System.out.println(res);
    }
    
    
    public static int get2(int x) {
        if (x % 2 != 0) return 0;
        int cnt = 0;
        while (x != 0) {
            if ((x & 1) == 1) cnt ++;
            x >>= 1;
        }
        return cnt;
    }
    
    public static int get5(int x) {
        if (x % 5 != 0) return 0;
        int cnt = 0;
        while (x % 5 == 0) {
            cnt ++;
            x /= 5;
        }
        return cnt;
    }
}
```
